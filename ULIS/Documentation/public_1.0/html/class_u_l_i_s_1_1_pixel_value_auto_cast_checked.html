Detailed Description

The PixelValueAutoCastChecked class provides a mean of auto-casting PixelValue.

By overloading the call operator(), a PixelValue can be retrieved automatically without explicit cast from the caller. The cast is checked with an assert which will fail if you try to assign the wrong type.
