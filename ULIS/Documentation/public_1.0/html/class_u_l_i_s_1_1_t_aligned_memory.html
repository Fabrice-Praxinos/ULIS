Detailed Description

template<typename T, size_t _Align>
class ULIS::TAlignedMemory< T, _Align >

[Deprecated] The TAlignedMemory template class provides a mean of managing arbitrary alignement for data in memory.

There are two template parameters:

  • The template type parameter T is the type of the data to allocate in an aligned fashion.
  • The template value parameter _Align of type size_t is the requested alignement ( any unsigned integer value, but more likely a power of 2, e.g: 8, 16, 32, 64, ... ).
