Detailed Description

template<int _S>
struct ULIS::TRegistry< _S >

The TRegistry struct provides a compile-time uint32 index registry.

It is acts like a vector of uint32, meant to hold CRC32 hashes, providing some kind of type registry and soft runtime type analysis for blocks.
