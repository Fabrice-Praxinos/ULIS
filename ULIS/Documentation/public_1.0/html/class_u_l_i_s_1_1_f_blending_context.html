Detailed Description

The FBlendingContext class provides a context for all Blending operations on Blocks.

The block on top will be blended on the back block, according to the specified parameters.

  • Warning ! Both blocks should be the same underlying format in order for the function to succeed.
  • If the two blocks do not have the same type, the function will fail with an assertion in debug builds.
  • In a release build, nothing prevents the function to run but it will end up corrupting memory and doing all sorts of crazy things.

The block on top will be blended on the back block, according to the specified parameters.

Parameters
iBlockTopThe pointer to the IBlock on top ( remains untouched ).
iBlockBackThe pointer to the IBlock on back ( receives the blend ).
iModeThe blending mode ( see eBlendingMode ).
iAreaThe area to blend in the back block.
iOpacityThe opacity used to perform the blend, beetween 0 and 1.
iPerformanceOptionsThe Performance Options for this operation, see FPerformanceOptions.
iCallInvalidCBWhether or not the function should call the invalid call back in the back block after the operation finished.
