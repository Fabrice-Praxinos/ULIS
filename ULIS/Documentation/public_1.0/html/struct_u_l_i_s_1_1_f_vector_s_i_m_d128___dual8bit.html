Detailed Description

[Deprecated] The FVectorSIMD128_Dual8bit struct provides wrapper around two FVectorSIMD128_8bit.

The FVectorSIMD128_Dual8bit is used for intermediate results when multiplying or adding multiple FVectorSIMD128_8bit.
