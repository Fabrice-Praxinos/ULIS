Detailed Description

The PixelProxyAutoCastChecked class provides a mean of auto-casting PixelProxy.

By overloading the call operator(), a PixelProxy can be retrieved automatically without explicit cast from the caller. The cast is checked with an assert which will fail if you try to assign the wrong type.
