Detailed Description

The FConversionContext class provides a context for Converting layout, types and colorspaces on Blocks.

static

Convert a block from source format and layout to the destination format and layout.

The source and destination can be any layout and type but they must share the same underlying color model.

  • Warning ! Source and destination must share the same width and height.
  • If the source and destination are the exact same type and layout, a copy of source into destination is performed instead.
    Parameters
    iBlockSrcThe pointer to the source IBlock, const, it will not be modified.
    iBlockDstThe pointer to the destination IBlock, it will receive the conversion result.
    iPerformanceOptionsThe Performance Options for this operation, see FPerformanceOptions.
